# A Simulation-Based Analysis of Channel Selection Algorithms for Cognitive Radio Networks

This repository contains the complete MATLAB simulation framework for the research paper exploring and comparing channel selection algorithms (CSAs) in a dynamic Cognitive Radio Network (CRN) environment.

## Abstract

The efficient utilization of the wireless spectrum remains a critical challenge, addressed promisingly by Cognitive Radio technology. A key determinant of CR network performance is the ability of Secondary Users to quickly identify vacant spectrum holes. This project presents a simulation-based investigation into CSAs, moving from heuristic-based approaches to a more robust, learning-oriented framework using Multi-Armed Bandits (MAB). We compare a baseline sequential-search **Epsilon-Greedy** algorithm against an advanced **Upper Confidence Bound (UCB1)** algorithm adapted for parallel sensing. Our results conclusively demonstrate that the intelligent, parallel UCB1 strategy significantly outperforms the sequential baseline across all key metrics: channel finding delay, switching rate, and overall network throughput.

## Project Structure

The repository is organized into a modular structure for clarity and ease of use:

-   `/simulation_core`: Contains the core simulation engine, including the environment model and the MAB algorithm implementations.
-   `/experiments`: Contains the main scripts used to run the experiments and generate the data for each figure.
-   `/plotting`: Contains the dedicated functions responsible for generating the final, publication-quality plots.
-   `/results`: Contains the final output graphs generated by the simulations.

## Methodology

### 1. The Multi-Armed Bandit (MAB) Framework

We model the channel selection task as a classic MAB problem, where each channel is an "arm" and finding it vacant is a "reward." This allows us to use robust, learning-based algorithms to manage the exploration-exploitation trade-off.

-   **Baseline (Sequential): Epsilon-Greedy Algorithm**
-   **Advanced (Parallel): UCB1 (Upper Confidence Bound) Algorithm**

### 2. The Simulation Environment

The project is built on a custom, event-driven simulator in MATLAB.
-   **Primary Users:** PU activity is modeled as a two-state Markov process, with the duration of ON (busy) and OFF (idle) states following an exponential distribution.
-   **Secondary Users:** SU behavior is driven by the arrival of data packets according to a Poisson process.
-   **Statistical Validity:** We use a Monte Carlo approach, averaging the results of many independent simulation runs to ensure statistically stable and meaningful conclusions.

## How to Run the Simulations

1.  **Prerequisites:** MATLAB (tested on version R2020a and later).
2.  **Clone the Repository:**
    ```bash
    git clone https://github.com/YourUsername/cognitive-radio-channel-selection.git
    ```
3.  **Run an Experiment:**
    -   Open MATLAB and navigate to the `cognitive-radio-channel-selection/` directory.
    -   Add the subfolders to the MATLAB path:
        ```matlab
        addpath(genpath(pwd));
        ```
    -   Open one of the scripts in the `/experiments` folder (e.g., `main_experiment_fig5.m`) and click the "Run" button.

The simulation will execute and produce the corresponding figure. **Note:** The Monte Carlo simulations are computationally intensive and may take several minutes to complete.
