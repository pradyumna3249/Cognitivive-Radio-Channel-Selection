# A Simulation-Based Analysis of Channel Selection Algorithms for Cognitive Radio Networks

This repository contains the complete MATLAB simulation framework for analyzing and comparing Channel Selection Algorithms (CSAs) in a dynamic Cognitive Radio Network (CRN) environment.

The study models intelligent spectrum access using a Multi-Armed Bandit (MAB) framework and demonstrates the superiority of parallel learning-based strategies over heuristic approaches.

---

## 📌 Abstract

Efficient wireless spectrum utilization remains one of the most critical challenges in modern communication systems. Cognitive Radio (CR) technology enables Secondary Users (SUs) to dynamically access vacant spectrum bands without interfering with Primary Users (PUs).

In this work, the channel selection problem is modeled as a Multi-Armed Bandit (MAB) problem. We compare:

- **Baseline:** Sequential-search Epsilon-Greedy Algorithm  
- **Advanced:** Parallel Upper Confidence Bound (UCB1) Algorithm  

Through extensive Monte Carlo simulations, we show that the intelligent parallel UCB1 strategy significantly outperforms the sequential baseline across key performance metrics:

- Channel finding delay  
- Channel switching rate  
- Network throughput

## 🧠 Methodology

### 1️⃣ Multi-Armed Bandit (MAB) Framework

- Each channel is modeled as an "arm"
- Successful detection of a vacant channel is treated as a "reward"
- Exploration–Exploitation trade-off handled via:

  - **Epsilon-Greedy (Sequential Baseline)**
  - **UCB1 (Parallel Learning-Based Strategy)**

---

### 2️⃣ Simulation Environment

The system is built using a custom event-driven MATLAB simulator.

**Primary Users (PUs):**
- Modeled as a two-state Markov process
- ON (busy) and OFF (idle) durations follow exponential distributions

**Secondary Users (SUs):**
- Packet arrivals follow a Poisson process
- Channel sensing and switching modeled explicitly

**Statistical Validation:**
- Monte Carlo averaging across multiple independent runs
- Ensures statistically stable and meaningful conclusions

# ▶️ How to Run the Simulations

## Prerequisites
- MATLAB (Tested on R2020a and later)

---

## Clone the Repository

```bash
git clone https://github.com/YourUsername/cognitive-radio-channel-selection.git
```

---

## Run an Experiment

1. Open MATLAB  
2. Navigate to the repository directory  
3. Add subfolders to the MATLAB path:

```matlab
addpath(genpath(pwd));
```

4. Run experiment scripts inside `/experiments`, for example:

```matlab
main_experiment_fig5
main_experiment_fig6
main_experiment_fig7_8
```

---

⚠️ **Note:**  
Monte Carlo simulations are computationally intensive and may take several minutes to complete.
